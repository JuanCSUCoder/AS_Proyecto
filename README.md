# Instrucciones ejecución

## Requerimientos de Software

- Docker
- Docker Compose
- NodeJS
- NPM

## Microservicios de Backend

En la raíz del proyecto:

```bash
docker compose up
```

## Frontend

```bash
cd /microservices/ms7-web
```

En `/microservices/ms7-web`:

```bash
npm i
```

```bash
npm run build
```

```bash
npm run start
```
